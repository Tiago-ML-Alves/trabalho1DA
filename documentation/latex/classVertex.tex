\subsubsection*{template$<$\mbox{\hyperlink{classEdge}{class}} \mbox{\hyperlink{classEdge}{T}}$>$\newline
class Vertex$<$ T $>$}
Represents a node in the graph. 

Stores adjacency lists (outgoing and incoming edges), auxiliary fields used by graph algorithms (BFS, DFS, Dijkstra, Tarjan SCC, topological sort) and the \doxylink{classMutablePriorityQueue}{Mutable\+Priority\+Queue} index required for Prim/\+Dijkstra.
