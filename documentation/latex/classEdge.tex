\subsubsection*{template$<$class T$>$\newline
class Edge$<$ T $>$}
Represents a directed weighted edge with optional flow and reverse pointer. 

Used both for standard graph traversal and for residual-\/graph operations in the Edmonds–\+Karp Max-\/\+Flow algorithm.
