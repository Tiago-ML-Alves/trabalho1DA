{\footnotesize\ttfamily \mbox{\hyperlink{classEdge}{void}} Parser\+::parse (\begin{DoxyParamCaption}\item[{\mbox{\hyperlink{classEdge}{const}} \mbox{\hyperlink{classEdge_ab976213419283ed754f42f57b195a288}{std\+::string}} \&}]{filename,  }\item[{\mbox{\hyperlink{classEdge_ab976213419283ed754f42f57b195a288}{std\+::map}}$<$ \mbox{\hyperlink{classEdge}{int}}, \mbox{\hyperlink{structSubmission}{Submission}} $>$ \&}]{submissions,  }\item[{\mbox{\hyperlink{classEdge_ab976213419283ed754f42f57b195a288}{std\+::map}}$<$ \mbox{\hyperlink{classEdge}{int}}, \mbox{\hyperlink{structReviewer}{Reviewer}} $>$ \&}]{reviewers,  }\item[{\mbox{\hyperlink{structParameters}{Parameters}} \&}]{parameters,  }\item[{\mbox{\hyperlink{structControl}{Control}} \&}]{control }\end{DoxyParamCaption})\hspace{0.3cm}{\ttfamily [static]}}

Iterates through every line of the file. Lines starting with {\ttfamily \#} switch the active section; all other non-\/empty lines are forwarded to the matching helper. Unknown {\ttfamily \#}-\/prefixed lines (that are not recognised section headers) are silently skipped so that comments inside the file are allowed.

\begin{DoxyNote}{Note}
The {\ttfamily submissions} and {\ttfamily reviewers} parameters are passed {\bfseries{by reference}} so that inserted entries are visible to the caller. (Previous version passed them by value — a bug that caused all parsed data to be lost.) 
\end{DoxyNote}
