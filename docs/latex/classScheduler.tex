\doxysubsection{Detailed Description}
Orchestrates Max-\/\+Flow execution and interprets the results. 

On construction\+: resets flows → runs Edmonds–\+Karp → extracts assignments. Call \doxylink{classScheduler_ae9fca17a227c2bf7050735da5327310a}{run\+Risk\+Analysis(k)} afterward to identify critical reviewers.
